# 🏥 Medical Insurance Cost Prediction

This Machine Learning project that predicts medical insurance charges based on customer information such as age, BMI, smoking status, number of children, sex, and region.

---

## 📌 Project Overview

Medical insurance costs vary depending on several factors, including age, BMI, smoking habits, and family size. This project uses Machine Learning techniques to predict medical insurance charges accurately. The project covers the complete machine learning pipeline, from data preprocessing and exploratory data analysis (EDA) to model training, hyperparameter tuning, and evaluation.

---

## 🎯 Objective

The main objective of this project is to develop a regression model that can accurately estimate medical insurance charges based on customer information.

---

## 📂 Dataset

The dataset contains the following features:

| Feature | Description |
|---------|-------------|
| Age | Age of the customer |
| Sex | Gender of the customer |
| BMI | Body Mass Index |
| Children | Number of dependent children |
| Smoker | Smoking status |
| Region | Residential region |

### Target Variable

- **Charges** – Medical insurance cost
